## Justification

This project took a long time to research, I had to research the following:

 - shader programming
 - Unity rendering API
 - academic rendering math papers
 - alternate methods to Gaussian Splatting
 - how to skip paywalls for Gaussian Splat assets
 - how to modify assets to trim away unwanted Gaussians

I know for a fact that I spent more than a full week struggling with various rendering bugs.
The process of writing shader code is extremely non-linear and can only be achieved from a deep understanding of both math and programming.

This time figure is most likely an underestimation.
